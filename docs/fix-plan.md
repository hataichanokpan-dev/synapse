#### Fix B1: SynapseService Bridge Class

**Problem**: MCP Server เรียก Graphiti โดยตรง ข้าม LayerManager ทั้งหมด

**Files Affected**:
- `synapse/mcp_server/src/graphiti_mcp_server.py` (MAJOR REFACTOR)
- `synapse/services/synapse_service.py` (NEW)
- `synapse/services/__init__.py` (NEW)

**Approach**:
```python
# synapse/services/synapse_service.py
class SynapseService:
    def __init__(self, config: GraphitiConfig):
        self.layer_manager = LayerManager(...)
        self.graphiti_client = ...

    async def add_memory(self, name: str, episode_body: str, ...):
        # 1. classify content → layer
        layer = self.layers.detect_layer(content)

        # 2. route to appropriate layer
        if layer == MemoryLayer.EPISODIC:
            self.layers.record_episode(content)
        elif layer == MemoryLayer.PROCEDURAL:
            self.layers.learn_procedure(trigger, steps)
        elif layer == MemoryLayer.USER_MODEL:
            self.layers.update_user(user_id, add_note=content)

        # 3. ALWAYS push to Graphiti for knowledge graph
        await self.graphiti.add_episode(name, content, source_desc)
```

**Risk**: High — Major architectural change
**Dependencies**: None (foundation fix)

#### Fix B5: User Isolation for Multi-User Support

**Problem**: Singleton pattern ทำให้ multi-user ไม่ได้ — ทุก managers ใช้ global singleton

**Files Affected**:
- `synapse/layers/manager.py`
- `synapse/layers/semantic.py`
- `synapse/layers/episodic.py`
- `synapse/layers/procedural.py`
- `synapse/layers/working.py`
- `synapse/layers/user_model.py`

**Current Pattern**:
```python
_manager: Optional[SemanticManager] = None

def get_manager(graphiti_client=None) -> SemanticManager:
    global _manager
    if _manager is None:
        _manager = SemanticManager(graphiti_client)
    return _manager
```

**Approach**:
```python
_managers: Dict[str, LayerManager] = {}

def get_layer_manager(user_id: str = "default") -> LayerManager:
    if user_id not in _managers:
        user_db_base = Path.home() / ".synapse" / user_id
        _managers[user_id] = LayerManager(db_base_path=user_db_base)
    return _managers[user_id]

# Result:
# user A → ~/.synapse/userA/episodic.db
# user B → ~/.synapse/userB/episodic.db
```

**Risk**: High — Affects all managers
**Dependencies**: B1 (part of SynapseService refactoring)
